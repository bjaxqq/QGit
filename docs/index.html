QGit - GitHub Repository Creator

Automate your Git workflow with one click

Prototype Notice

This is a proof-of-concept application developed through Quinnipiac University's partnership with Microsoft. Not for commercial use.

Key Features

  • ✓ Convert local projects to Git repos
  • ✓ Automatic .gitignore generation
  • ✓ Custom README.md creation
  • ✓ GitHub repo mirroring
  • ✓ Public/private visibility options
  • ✓ Direct URL output

Requirements

Software

  • Java
  • Git

Account

  • GitHub account
  • Personal access token

Requires external libraries: GitSubprocessClient.jar and GitHubApiClient.jar
